# The Met Landing Page

A responsive landing page for The Metropolitan Museum of Art, built to showcase the museum’s atmosphere, visit information, and key highlights in a modern, user-friendly layout.

## Live Preview

A public demo is not currently available. Once the project is deployed, update this section with the live URL.

## Design Reference

This project is inspired by the The Met landing page design and follows a clean, minimalist visual approach.

## Technologies Used

- HTML5
- SCSS
- JavaScript
- Vite
- Cypress

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/chlodav/LandingPage_Projects_NB.git
cd LandingPage_Projects_NB
```

2. Install dependencies:

```bash
npm install
```

3. Run the project locally:

```bash
npm start
```

The app will be available in your browser at the local Vite address shown in the terminal.

## Features

- Responsive layout for desktop and mobile screens
- Interactive mobile navigation menu
- Clear museum hours and contact information
- Modern styling with SCSS and reusable components
